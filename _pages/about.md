**RunPas** (Run + Pas = pas en català) és una web per a un grup de corredors.

## Funcionalitats

- Registra les teves curses (distància, temps, GPX)
- Consulta la classificació per categories: 5K, 10K, Mitja Marató, Marató, Trail, Ultra
- Blog d'entrenament
- Sync amb Strava
- Estadístiques personals

## Tecnologia

Construït amb Jekyll + Chirpy, Supabase i GitHub Pages.
